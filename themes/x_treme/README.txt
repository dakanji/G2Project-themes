The x_treme theme comes with its own colorpacks, imageframes, icon sets and
an improved slideshow template.

Installation instructions:
-------------------------
1. Install the Gallery colorpack, icons, imageframe and slideshow module.
2. Copy the contents of themes/x_treme/colorpacks/ to 
   modules/colorpack/packs/
3. Copy the contents of themes/x_treme/imageframes/ to
   modules/imageframe/frames/
4. Copy the contents of themes/x_treme/iconpacks/ to
   modules/icons/iconpacks/
5. Copy themes/x_treme/templates/Slideshow.tpl to
   modules/slideshow/templates/local/Slideshow.tpl
6. Copy themes/x_treme/templates/RatingImagePreload.tpl to
   modules/rating/templates/local/RatingImagePreload.tpl
7. Copy the contents of themes/x_treme/comment/ to
   modules/comment/

Now you can select all colorpacks, image frames and icon sets in your
x_treme theme settings in 'Site Admin -> Themes.'
